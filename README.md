# wf-magma-hmagma

## Files for running MAGMA or H-MAGMA analysis workflows on Terra
- *Dockerfile*: containerize MAGMA and install dependencies
- *cloudbuild.yaml*: build and push docker image to Google Artifact Registry via Cloud Build
- *magma-annotate.wdl*, *magma-gene.wdl*, *magma-set.wdl*: define workflow corresponding to each MAGMA analysis step
- *.dockstore.yml*: sync workflows to Dockstore for import to Terra
- *data/*: miscellaneous files for runnning MAGMA, generated by scripts in *data-scripts/*

## Approximate run time
- Cloud Build: a few minutes
- Dockstore: a few minutes
- *magma-annotate* workflow on Terra: a few minutes
- *magma-gene* workflow on Terra: variable, most <1 hour
- *magma-set* workflow on Terra: a few minutes
